Geomys

I recently wrote a gopher server in Rust using rotor. Gopher is an interesting net protocol of old that can be considered a predecessor to HTTP. It has been largely abandoned, however a number of enthusiasts still run servers, such as z3bra and ix.

After reading about the protocol and looking at what other people were doing, I decided to set up my own server. I thought that writing a gopher server would be a good project to learn more about TCP and rotor without spending too much time. I also learned how to follow a specification from this project. I decided to build it on top of rotor because, other than mio, it is the only library I know of in Rust for working with TCP. Rotor also has the advantage of async, which is always welcome.
